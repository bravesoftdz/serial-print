Описание программы

Программа предназначена для печати серийных номеров и другой переменной информации на театральных, лотерейных, проездных и прочих билетах, листовках, бланках.

Программа содержит базу данных макетов типографских листов с билетами, макетов билетов, наборы нумераторов.

Начало работы

  • Добавление нового проекта
  • Настройка нумераторов
  • Настройка шаблонов билетов
  • Настройка шаблонов листов
  • Создание плана нумерации
  • Создание плана зала
  • Настройка принтеров

Добавление нового проекта

На закладке "Проекты" находится список проектов. Проект может содержать под-проекты.

Для добавления нового проекта вызываем контекстное меню списка проектов (правой кнопкой мыши на списке проектов) и выбираем "Добавить". Будет создан новый проект. В подробностях проекта нужно указать название проекта и его описание. В описании можно указывать любую информацию, относящуюся к проекту - примечания, заметки, итд..

Для добавления под-проекта нужно выбрать основной проект и в контекстном меню (правая кнопка мыши) выбрать "Добавить подчиненного". Подчиненные проекты ничем не отличаются от основных, кроме расположения в списке проектов.

Настройка нумераторов

Список нумераторов выбраного проекта расположен на закладке "Шаблоны". Для добавления нового нумератора в контекстном меню списка нумераторов (правая кнопка мыши) выбираем "Добавить". После добавления нумератора ему нужно дать соответствующее название, например "Ряд" или "Место", указать начальное значение и основное действие.

Настройка принтеров

Настройка геометрии билетов

1. Определяется шаблон листа (размер, свойства печати, переплет, итд..) 2. Печать координатной сетки на пробном листе 3. Определение числа билетов на листе, на каждый билет заводится карточка билета 4. Установка координат верхнего левого угла билета для каждой карточки билета 5. Определяются шаблоны билетов - размер билета, свойства для планирования нумерации 6. Определяются серии номеров, их положение внутри шаблона билета или листа

Планирование нумерации

С помощью мастера планирования создается план нумерации - набор правил и формул установки номеров, в зависимости от номера листа и положения билета. Простейший план нумерации включает в себя определение серии номеров - начальный номер, шаг, диапазон, формат номера, префикс. Сложная серия может включать в себя доп-серии, формулы и условия.

Для планирования нумерации билетов для зала с сиденьями, используется конструктор зала. В нем рисуется контур зала и вставляются "места" - прямоугольники с номером места. Можно отметить несколько мест и назначить им общий номер или название ряда/сектора/стороны. Для отдельных мест можно задавать пропуск (место неисправно или зарезервировано) или особые признаки, используемые в плане нумерации.

Для учета особенности геометрии листов, в формулах плана нумерации можно использовать разные шаблоны листов под разными именами. Готовый план нумерации можно сохранить в файл формата XML или TXT, или прочитать из файла.

Для контроля результата имеется режим предварительного просмотра, в котором отображаются контуры билетов и номера и дополнительная информация (название шаблона листа, особые признаки билетов). В режиме печати контуры билетов и дополнительная информация не печатаются.
